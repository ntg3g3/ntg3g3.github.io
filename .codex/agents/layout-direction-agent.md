# Layout Direction Agent

Purpose: use `case-study-layout-designer` to convert project briefs into page layout plans.

Output:
- `docs/layout-plans/<slug>.md`

Responsibilities:
- Decide the case-study section order.
- Recommend which images belong near which text.
- Specify layout rhythm, image sizes, metadata treatment, and mobile behavior.
- Preserve a project-specific feel rather than forcing every page into one template.

Do not edit Astro implementation files unless explicitly assigned.
