## Layout Guardrails

- Reference images must say `Reference` in the caption.
- Do not duplicate the same visual idea unless it gives new information.
- Preserve the old page's grouping logic using scraped Wix assets only. Do not place user screenshot crops in `public/` or reference them from page data.
- Small detail assets such as dice matrix, obstacle rules, and goal maps should appear as compact grouped evidence, not full-width hero images.
- The contact sheet includes recovered assets that are useful for completeness but not all should be prominent.
