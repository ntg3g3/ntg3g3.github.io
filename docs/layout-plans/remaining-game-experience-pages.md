Status: approved baseline pass for sitewide review.

Scope:

- `remember-me`

- `wristx`
- `roominate`

## Intent

These pages should pass the new portfolio standard without being over-promoted above Dice Adventure, CAVERN, Project Amazon, Skyweaver, or future public-safe Last Trigger material.

The correct treatment is a compact case-study baseline:
- preserve the old Wix narrative order captured in each project brief
- keep text short and contextual
- use the strongest recovered image group as the page's evidence
- avoid inventing new role or implementation claims
- avoid turning every page into the same long left-text/right-image template

## Shared Layout Standard

Hero:
- Project title, role/category, one short summary.
- Cover or strongest visual from the original page.

Body:
- Two to five visual sections depending on evidence.
- Dense diagrams can be wide.
- Small UI, prop, or character assets should remain in gallery/pair layouts.
- Motion recovered from Wix should stay visible where it explains interaction.

Copy:
- One short body paragraph per section at most.
- No new team claims unless the original page or user confirmation supports them.
- Captions should identify what the viewer is looking at, not over-explain.

## Acceptance Criteria

- Every page in scope has a brief with original Wix logic.
- Current implementation has at least one intentional visual section per page.
- Images are grouped by meaning instead of dumped as a single gallery.
- Text density stays below priority case-study pages.
- Build, image, route, and mobile checks pass.
