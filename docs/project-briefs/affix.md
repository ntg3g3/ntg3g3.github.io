## System / Gameplay / Interaction Model

This is a product/system project. The interaction model is assembly:

- User aligns a furniture panel or component with a base/frame.
- A connector or sliding element links the component to the structure.
- The same connection principle can be reused across different furniture parts.
- The product can be explained from whole object to detail: final furniture, cutaway connection, exploded/assembly diagram, connector specification.

The exact locking action is inferred from the diagrams; do not overstate whether it is snap-fit, sliding, magnetic, or screwless unless confirmed.
