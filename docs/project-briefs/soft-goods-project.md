## System / Gameplay / Interaction Model

This is a wearable/product interaction:

- Vest mode: the object sits on the body with straps and pockets accessible from the front.
- Transformation: fabric panels fold or gather, straps reposition, and the vest body becomes a tote volume.
- Tote mode: the same object carries items as a bag, with handles/straps exposed.
- Detail use: pockets and closures support storage and access.

The transformation steps are inferred from diagrams and photos. Do not write a precise step-by-step instruction unless confirmed by Yutao.
