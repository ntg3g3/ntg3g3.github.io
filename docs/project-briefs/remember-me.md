## Original Wix Page Logic

The original page uses a process-to-final structure. It begins with hand sketches and notebook images, which makes sense because the project is about memory and story. The sketches are not throwaway process images; they establish the emotional premise and the motif of memories appearing as fragments.

The page then moves into tangible props and 3D models: remote, popsicle, keys, glasses, and other everyday objects. This middle section explains how the story becomes AR interaction. Character model images bridge process and prototype by showing digital figures placed beside original sketches. The final title and animated AR loop serve as the outcome.

The rebuild should keep the page quiet and image-led. Text should not over-explain Alzheimer's; it should explain that everyday objects become memory cues, then let sketches and AR scenes do the emotional work.
